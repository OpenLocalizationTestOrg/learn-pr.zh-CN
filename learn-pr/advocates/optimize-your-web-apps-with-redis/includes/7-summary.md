Redis 缓存可提高依赖数据库的系统的性能和可伸缩性。 通过将频繁访问的数据暂时复制到靠近应用程序的快速存储中来提高性能。 对于 Redis 缓存, 此快速存储位于内存中, 包含 Redis 缓存, 而不是由数据库从磁盘加载。
